# Description
Our website is a simplified tier list creator (inspired by tiermaker.com). Users cain sign up/login, create tier lists through an interactive editor, save them to their account, and view them later on a personal profile page. Other users can browse/search for tier lists and view creators’ profiles to see the tier lists they’ve made.
# Install Guide
To install our website, follow these instructions.

1) Clone the repository from Github in which the website resides in by typing the following in your terminal:
`git clone git@github.com:Araf215/2Yellow2Brown_aoanulh2789_owenz20_yuhangp_kalimulk.git`

2) Open a virtual environment for the website by typing the following:
`python3 -m venv venv`

3) Activate that virtual environment:  
Windows: `venv\Scripts\activate`  
MacOS and Linux: `. venv/bin/activate`

4) Install the website components:  
`pip install -r 2Yellow2Brown_aoanulh2789_owenz20_yuhangp_kalimulk/requirements.txt`

# Launch Codes
This is the command to launch the website:
`python 2Yellow2Brown_aoanulh2789_owenz20_yuhangp_kalimulk/app/__init__.py`

Then, enter this URL in your browser:  
`http://127.0.0.1:5000`
